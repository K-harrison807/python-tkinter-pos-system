# python-tkinter-pos-system

# Description

This is a desktop-based Point of Sale (POS) system built using Python, Tkinter for the graphical user interface, and SQLite for database management. The application demonstrates full CRUD operations, inventory management, and transaction calculations.

# Features

Add new products

Update product details

Delete products

Filter/search products

Inventory management

Transaction total calculations

SQLite database integration

User-friendly GUI built with Tkinter

# Technologies Used

Python

Tkinter

SQLite

# Project Focus

This project focuses on desktop application development, database integration, and implementing structured CRUD functionality with a graphical interface.

# How to Run

Install Python (version 3.x).

Ensure Tkinter is installed (usually included with Python).

Run the main Python file:

python main.py
